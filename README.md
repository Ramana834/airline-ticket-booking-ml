# ✈️ Intelligent Airline Ticket Booking & Management System

> A smart airline ticket booking and management system powered by Machine Learning — built with Python & Django

---

## 🙋‍♂️ About This Project

This project is an intelligent airline ticket booking system that uses Machine Learning to provide smart features like price prediction, flight recommendations, and demand forecasting.

## ✨ Key Features

- ✅ Smart flight search with filters (source, destination, date)
- ✅ ML-based ticket price prediction
- ✅ Flight recommendations based on user preferences
- ✅ User registration & secure login
- ✅ Booking management (book, cancel, view history)
- ✅ Real-time flight data with domestic & international routes
- ✅ Hotel & airport information integration
- ✅ Emergency contact management

## 🚀 Setup & Installation

```bash
# Clone the repository
git clone https://github.com/Ramana834/airline-ticket-booking-ml.git
cd airline-ticket-booking-ml/airline_ticket_ml

# Create virtual environment
python -m venv env
env\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start server
python manage.py runserver
```
